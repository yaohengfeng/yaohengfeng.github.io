Hugo怎么搭建博客

January 6, 2020

一、 Hugo的下载

工欲善其事，必先利其器 ，我们要Hugo搭建博客，首先应该把Hugo下载好，下载地址，进入选择适合自己电脑的版本。

二、具体操作

进入Hugo文档按照文档里面的操作就可以配置完成。详细步骤可以借鉴下列的步骤

  1. 进入下载解压好的Hugo文件中，将hugo.exe 移到D:\Software\hugo文件夹中，并把D:\Software\hugo添加到环境变量path中，在当前目录打开git Bash，运行hugo version，得到版本号，即配置成功。

    如下图

  2. 然后进入Hugo官网

  1. 在git Bash 执行 hugo new site “文件名(最好是自己的GitHub地址加一个单词，如： ***@github.io_creater)”
  2. 进入这个目录(cd ***@github.io_creater)，执行git init和 git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 后，就会开始下载主题，如图：

  1. 下载完成后，执行 echo ‘theme = “ananke”’ » config.toml

  2. hugo new posts/“博客文章的名字（如：我的第一篇博客）".md

  3. 运行 hugo server -D ，就出现一段话，用浏览器打开地址(http://localhost:1313)，如图：

  4. 现在本地博客差不多建好了，就要开始上传到GitHub上了。
